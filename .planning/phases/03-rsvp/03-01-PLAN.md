Create `Assets/App/Scripts/UI/GuestRowUI.cs` as a standard MonoBehaviour following the project's `#region` structure (see CONVENTIONS.md). This component is attached to the guest row prefab that the developer creates in the Unity Editor.

**File structure (regions in order):**
1. `#region Inspector Variables` — all `[SerializeField] private` fields with `[Tooltip]`
2. `#region Properties` — read-only public accessors
3. `#region Public Methods` — `Initialize(string guestName)` only

**Inspector fields:**
```csharp
[Tooltip("Label displaying the guest's pre-filled name.")]
[SerializeField] private TMP_Text nameLabel;

[Tooltip("Toggle: guest IS attending. Part of an attendance ToggleGroup.")]
[SerializeField] private Toggle attendingToggle;

[Tooltip("Toggle: guest is NOT attending. Part of an attendance ToggleGroup.")]
[SerializeField] private Toggle notAttendingToggle;

[Tooltip("Meal toggle: meat (carne). Set isOn = true by default in prefab Inspector.")]
[SerializeField] private Toggle meatToggle;

[Tooltip("Meal toggle: fish (pesce).")]
[SerializeField] private Toggle fishToggle;

[Tooltip("Meal toggle: vegetarian (vegetariano).")]
[SerializeField] private Toggle vegetarianToggle;
```

**Properties:**
```csharp
public string GuestName => nameLabel != null ? nameLabel.text : string.Empty;
public bool IsAttending => attendingToggle != null && attendingToggle.isOn;
public string SelectedMeal
{
    get
    {
        if (meatToggle != null && meatToggle.isOn)           return "carne";
        if (fishToggle != null && fishToggle.isOn)           return "pesce";
        if (vegetarianToggle != null && vegetarianToggle.isOn) return "vegetariano";
        return "non specificato"; // fallback — prefab default (meatToggle.isOn) prevents this
    }
}
```

**Public method:**
```csharp
/// <summary>Pre-fills the guest name label. Called by RSVPPopup when instantiating rows.</summary>
public void Initialize(string guestName)
{
    if (nameLabel != null)
        nameLabel.text = guestName;
}
```

**Required usings:** `TMPro`, `UnityEngine`, `UnityEngine.UI`

**No Unity lifecycle methods** (no Awake/Start) — this is a pure data accessor component. The developer configures default Toggle states (meatToggle.isOn = true, attendingToggle.isOn = true) directly in the prefab Inspector.
